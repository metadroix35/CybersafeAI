##  Getting Started

To run locally:

1. Clone the repository  
   ```bash
   git clone https://github.com/metadroix35/CybersafeAI.git
   cd CybersafeAI
  ```

2. Install dependencies

   ```bash
   npm install
   ```

3. Run development server

   ```bash
   npm run dev
   ```

Any changes made via Lovable will also update this repo automatically.
